# Time-set

![time-set-image](/assets/images/timeset-cut.jpg)
> Web App with time functionality: 
> - mechanical clock 
> - digital clock 
> - current date, year 
> - set mech clocks (different time zones)
> - stopwatch
> - timer
> - custom calendar
